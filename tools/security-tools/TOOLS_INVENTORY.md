## Installation

Install all tools:

```bash
sudo /opt/synos-tools/scripts/install-all.sh
```

Install by category:

```bash
sudo /opt/synos-tools/scripts/install-category.sh web-application
```

Verify installation:

```bash
sudo /opt/synos-tools/scripts/verify-tools.sh
```
